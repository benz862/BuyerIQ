# BuyerIQ V2 Product Build Specification

BuyerIQ helps home buyers objectively evaluate properties before making one of the largest financial decisions of their lives.

BuyerIQ is not a real estate CRM. BuyerIQ is a buyer decision intelligence platform.

## Plans

- BuyerIQ Standard: $99 one-time purchase, own forever.
- BuyerIQ Pro: $149 one-time purchase, own forever, with advanced reports, regional intelligence packs, enhanced comparisons, and future feature updates.

## Core Workflow

1. User creates an account.
2. User chooses Buying, Renting, or Relocating mode.
3. User creates a property project manually.
4. BuyerIQ tracks questions, risks, timeline events, completeness, and regional concerns.
5. User reviews insights and a professional report.
6. User compares saved properties and exports data.

## BuyerIQ Score

Each property receives a 0-100 score across:

- Property Fit Score
- Risk Score
- Lifestyle Score
- Cost Score
- Confidence Score

Recommendation levels:

- 90-100: Excellent Match
- 80-89: Strong Candidate
- 70-79: Proceed With Caution
- 60-69: High Risk
- Below 60: Not Recommended
